# OpenClaw 项目组目录

该目录作为项目组根目录：`openclaw/`。

## 当前项目

### `practical-tools-wechat-console`
自媒体与 OpenClaw 运维一体化控制台：
- OpenClaw 配置/技能/记忆管理
- 飞书/公众号/小红书内容栏
- 公众号“实用软件/免费工具”领域推荐与关键词检索

项目入口：
- `openclaw/practical-tools-wechat-console`

快速启动：
```bash
cd openclaw/practical-tools-wechat-console
npm install
npm start
```
